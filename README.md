# webapp
This application is developed for skills improving without commercial (so far) background. As for now, project is free to be modified by any member of the `multiheaded` team.

## Wiki
Small wiki for this project is maintained by the whole team. Wiki contains articles, schemes, scripts etc that should bot be forgotten.
You can start reading wiki from  [home page](https://bitbucket.org/multiheaded/webapp/wiki/Home).

## Docker
Development and production versions of services are launched in docker container through docker images built individually for every project.
More info about scripts, launching and *why this is happening to me* could be found in the wiki.
